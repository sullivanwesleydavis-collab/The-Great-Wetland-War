# The-Great-Wetland-War
Fun water-like game featuring a pixelated swamp level.

## How to Play
- Run `python3 -m http.server 8000` in the project directory
- Open http://localhost:8000 in your browser
- Use arrow keys to move your green frog character around the swamp
- Press SPACEBAR to shoot yellow projectiles at enemy red frogs
- Avoid contact with enemy frogs (they damage you over time)
- Defeat all enemy frogs to win the level
- If your health reaches 0, you lose
- Press R to restart after winning or losing

## Objective
Survive the wetland battlefield and eliminate all enemy frog invaders!

## Features
- Procedurally generated swamp levels with water features, vegetation, and trees
- Player frog character (green 2x2 sprite) with health system
- Enemy frog units (red 2x2 sprites) with AI movement towards player
- Projectile combat system - shoot yellow projectiles with SPACEBAR
- Collision-based damage (touching enemies hurts you)
- Win/lose conditions (defeat all enemies or lose all health)
- Score system for defeated enemies
- Atmospheric fog effects
- Pixelated retro game style
