# The-Great-Wetland-War
Fun water-like game featuring a pixelated swamp level.

## How to Play
- Run `python3 -m http.server 8000` in the project directory
- Open http://localhost:8000 in your browser
- Use arrow keys to move your orange player character around the swamp
- Avoid trees and deep water - you can only walk on grass, mud, and shallow water
- Explore the randomly generated wetland terrain with scattered enemy frogs (red pixels)
- The camera follows your movement automatically

## Features
- Procedurally generated swamp levels with water features, vegetation, and trees
- Player character with collision detection
- Enemy units scattered throughout the level
- Atmospheric fog effects
- Pixelated retro game style
