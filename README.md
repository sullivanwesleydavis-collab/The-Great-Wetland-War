# The-Great-Wetland-War
Fun water-like game featuring a pixelated swamp level.

## How to Play
- Run `python3 -m http.server 8000` in the project directory
- Open http://localhost:8000 in your browser
- Use arrow keys to move your green frog around the swamp
- Avoid trees and water - you can only walk on grass and mud
- Explore the randomly generated wetland terrain

## Objective
Navigate through the swamp as a frog, exploring the pixelated wetland environment!

## Features
- Procedurally generated swamp levels with water features, vegetation, and trees
- Player frog character with collision detection
- Atmospheric pixelated graphics
- Simple exploration gameplay
